# 🐾 CAPYLULU 桌宠 浏览器扩展

> 可爱的 CAPYLULU 桌宠会在你打字时来回走动，停止打字时也会停下！

![CAPYLULU](icons/icon128.png)

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 🚶 **打字行走** | 在任何页面的输入框中打字时，CAPYLULU 会自动走动 |
| ⏸️ **自动停止** | 停止打字后约 0.8 秒，桌宠会自动停下进入待机状态 |
| 🖱️ **自由拖拽** | 可以拖拽桌宠到页面任意位置 |
| 💬 **点击互动** | 单击显示随机对话气泡 + 粒子特效；双击触发旋转动画 |
| ⚙️ **可调设置** | 通过扩展弹窗调整行走速度、停止延迟、大小等 |
| 🎨 **精美动画** | 行走弹跳、手臂摆动、待机摇晃/眨眼/浮动、粒子特效 |
| 🔄 **模型切换** | 内置多款预设模型，支持上传自定义 SVG 模型 |

## 📦 安装方法

### Chrome / Edge / Brave（Chromium 内核浏览器）

1. **下载扩展包** → 解压得到 `capylulu-pet-extension` 文件夹
2. 打开浏览器地址栏输入：`chrome://extensions/`
3. 开启右上角 **「开发者模式」**
4. 点击 **「加载已解压的扩展程序」** 按钮
5. 选择解压后的 `capylulu-pet-extension` 文件夹
6. ✅ 安装完成！CAPYLULU 会出现在所有网页上

### Firefox

1. 打开 `about:debugging#/runtime/this-firefox`
2. 点击「临时载入附加组件」
3. 选择 `manifest.json` 文件
4. ✅ 完成！（Firefox 需要稍作调整 manifest 才能正式发布）

## 🎮 使用说明

- **打字**：在任何输入框/文本区域/编辑器中输入文字 → CAPYLULU 开始走动
- **停止打字**：0.8秒后自动停下，进入待机浮动模式
- **拖拽移动**：按住桌宠拖动到任意位置
- **单击**：显示随机可爱对话 + 粒子爆发效果
- **双击**：旋转特效 + 双倍粒子
- **设置面板**：点击浏览器工具栏的 CAPYLULU 图标打开设置

## 🔧 设置选项

| 选项 | 说明 | 默认值 |
|------|------|--------|
| 模型选择 | 选择预设模型或上传自定义 SVG | CAPYLULU |
| 打字行走 | 是否在打字时启用走动 | ✅ 开启 |
| 移动方式 | 自由漫游 / 水平移动 / 原地待机 | 自由漫游 |
| 行走速度 | 1-8 级速度 | 3 |
| 停止延迟 | 停止打字后多久停下 (ms) | 800ms |
| 桌宠大小 | 小/中/大/超大 | 中 (80px) |

## 🛠️ 技术细节

- **Manifest Version**: V3（最新标准）
- **注入方式**: Content Script 自动注入到所有 `<all_urls>`
- **动画引擎**: CSS Keyframes + requestAnimationFrame 混合驱动
- **角色渲染**: 纯 SVG 绘制（无需外部图片资源）
- **兼容性**: Chrome 88+, Edge 88+, Brave, Firefox（需微调）
